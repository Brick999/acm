Problem 1: Redundant Paths [Harry Wiggins/SACO, 2005]

     Для того, чтобы попасть с одного из F (1 <= F <= 5,000) 
пастбищ (пронумерованных 1..F) на другое, Бесси и другие коровы 
стада должны пройти мимо Tree of Rotten Apples. Коровам надоело 
ходить все время по одной и той же дороге, и они хотят протоптать 
новые тропинки так, чтобы у них всегда было, как минимум, два 
отдельных маршрута между двумя любыми пастбищами. 
     Маршруты называются отдельными, если они не используют ни 
одной общей тропинки, даже если они посещают одни и те же 
промежуточные пастбища.
      Сейчас у них есть, как минимум, один маршрут между любыми
двумя парами пастбищ. А теперь они хотят иметь как минимум два таких
маршрута. Конечно, они могут двигаться только по протоптанным 
тропинкам при перемещении от одного пастбища к другому.
    Вам дано описание текущего множества R (F-1 <= R <= 10,000)
тропинок, каждая из которых соединяет ровно два различных пастбища.
Определите минимальное количество новых тропинок (каждая из которых
соединит ровно два пастбища), которые необходимо протоптать для 
того, чтобы получить как минимум два отдельных маршрута между любой 
парой пастбищ.
    Может быть более одной тропинки между двумя пастбищами, и Вы 
можете "топтать новую тропинку" для пары пастбищ, которые уже 
соединены тропинкой.

PROBLEM NAME: rpaths

INPUT FORMAT:

* Строка 1: Два разделенных пробелом целых числа F и R

* Строки 2..F+1: Каждая строка содержит два разделенных 
                 пробелом целых числа - номера пастбищ,
                 соединенных тропинкой. 

SAMPLE INPUT (файл rpaths.in):

7 7
1 2
2 3
3 4
2 5
4 5
5 6
5 7

INPUT DETAILS:

Один из способов визуализации тропинок представлен ниже:
   1   2   3
   +---+---+  
       |   |
       |   |
 6 +---+---+ 4
   |  / 5
   | / 
   |/ 
 7 +

OUTPUT FORMAT:

* Строка 1: Одно целое число - количество новых тропинок,
            которые нужно протоптать

SAMPLE OUTPUT (файл rpaths.out):

2

OUTPUT DETAILS:

Протоптав тропики от 1 к 6 и от 4 к 7 мы удовлетворим требования 
задачи.
   1   2   3
   +---+---+  
   :   |   |
   :   |   |
 6 +---+---+ 4
   |  / 5  :
   | /     :
   |/      :
 7 + - - - - 
Проверьте некоторые маршруты:
1 - 2:  1 -> 2            и   1 -> 6 -> 5 -> 2
1 - 4:  1 -> 2 -> 3 -> 4  и   1 -> 6 -> 5 -> 4
3 - 7:  3 -> 4 -> 7       и   3 -> 2 -> 5 -> 7
В действительности каждая пара пастбищ соединена двумя 
маршрутами.

Problem 2: Roping the Field [Hal Burch, 2004]

    Фермер Джон - истинный художник: он часто выполняет 
произведения искусства у себя на ферме. Сегодня, например, он хочет 
сконструировать на пастбище гигантскую "паутину".
Пастбище ФД - это большой выпуклый многоугольник с заборами вдоль 
границ и столбами в каждом из N углов (1 <= N <=150). Чтобы сделать 
паутину, ФД хочет навесить прямолинейно как можно больше веревок между
двумя не-соседними столбами так, чтобы никакие две веревки не 
пересекались.
    Есть одно усложнение - поле ФД не полностью пригодно. "Дьяволы"
создали G (0 <= G <= 100) кругов, все радиусом R (1 <= R <= 100,000),
через которые ФД не может протягивать веревки, даже не может 
касаться веревкой соответствующей окружности. Заметим, что несмотря 
на то, что центры всех кругов находятся внутри поля, радиусы могут 
быть настолько большими, что внутри кругов могут оказаться и столбы, 
и заборы.
    По заданным координатам столбов и кругов, определите 
максимальное количество веревок, которое ФД может использовать для 
создания своей "паутины".
    Координаты столбов и центров кругов - пары целых чисел, каждое 
из которых в диапазоне 0..1,000,000.

PROBLEM NAME: roping

INPUT FORMAT:

* Строка 1: Три разделенных проблелом целых числа: N G R

* Строки 2..N+1: Каждая строка содержит два разделенных пробелом
            целых числа - X Y координаты столба поля ФД

* Строки N+2..N+G+1: Каждая строка содержит два разделенных пробелом
            целых числа - X Y координаты центра круга внутри поля ФД

SAMPLE INPUT (файл roping.in):

5 3 1
6 10
10 7
9 1
2 0
0 3
2 2
5 6
8 3

INPUT DETAILS:

Поле в виде пятиугольника, в котором все возможные веревки 
блокированы тремя кругами, за исключением веревки между столбами 
2 и 4.

OUTPUT FORMAT:

* Строка 1: Одно целое число - максимальное количество веревок,
            которое может использовать ФД.

SAMPLE OUTPUT (файл roping.out):

1

Problem 3: Corral the Cows [Marcello Herreshoff, 2005]

    Фермер Джон хочет построить загон для своих коров. Загон должен 
быть в форме квадрата и содержать не менее C (1 <= C <= 500) пастбищ.
Владения ФД включают всего N (C <= N <= 500) пастбищ, каждое 
размером 1 х 1 и целыми X и Y координатами левого нижнего угла 
(X и Y в диапазоне 1..10,000). Иногда более одного пастбища может 
иметь одни и те же координаты, тогда на вводе такие координаты могут
появится два или более раз. Загон содержит пастбище, если оно целиком
находится внутри границ загона.
     Помогите ФД вычислить минимальную длину стороны квадрата, 
который будет содержать C пастбищ.

PROBLEM NAME: corral

INPUT FORMAT:

* Строка 1: Два разделенных пробелом целых числа: C и N

* Строки 2..N+1: Каждая строка содержит два разделенных пробелом
            целых числа - X и Y координаты пастбища.

SAMPLE INPUT (файл corral.in):

3 4
1 2
2 1
4 1
5 2

INPUT DETAILS:

|*   *
| * *
+------

OUTPUT FORMAT:

* Строка 1: одно целое число - длина одной стороны квадрата 
            минимального размера, содержащего не менее C пастбищ.

SAMPLE OUTPUT (файл corral.out):

4

OUTPUT DETAILS:

Ниже представлено одно из решений 4х4.
Символы C показывают поверхность загона.
Существует и множество других решений.

|CCCC
|CCCC
|*CCC*
|C*C*
+------
